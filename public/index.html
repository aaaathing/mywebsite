History

From thingmaker.

Early days of thingmaker's website

The website started as many files and folders on a old computer at somewhere between late 2020 to early 2021. At this time, it was not accessible from the internet as it was local files. This was also when thingmaker started adding things to MineKhan. Some time before December 2020, he downloaded a version of MineKhan (0.7.1) originaly from some time before this. The first things added to it were textures and new blocks called 'soup'. Later adding redstone was tried, by adding on and off blocks, then a block called 'buffer' was added which was supposed to behave like a redstone repeater. The redstone blocks didn't work well but stayed. Some time, later, cows and pigs were added which move in random directions and looked bad. Starting near April 2021, thingmaker checked the profile of RedMonster (https://www.khanacademy.org/profile/monsterred/projects) frequently for new things to add to the version of MineKhan. On May 8 2021, the version of MineKhan with changes by thingmaker was uploaded to R____t. A few days later, on May 12 2021, the website was also moved onto R____t (https://replit.com/@thingMaker/www) and became accessible from the internet. The website was still very basic. Soon, on May 24 2021, a server (https://replit.com/@thingMaker/server) was made for the website. A simple login were added to the website and server. The code for the login was copied from somewhere else because thingmaker did not know how to make a login at that time. Websockets were also experimented with because they would allow adding multiplayer to thingmaker's version of MineKhan. In May 2021, multiplayer was added to the version of MineKhan. Posts and comments on posts was added to the server and website. Posts were a part of the website where people could post text. Then, the first post not by thingmaker appeared. (name: Hello, by: Spider, content: Hello) After some time, there were more new users, including: Roger999, sup6666, MyHat3, Pillow1007 In June 2021, profile pictures were added to the website and server. Notifications were also added and a way to ban in MineKhan was added to the server. Live comments were also added to posts. You can also see list of updates here and old versions here. This was also when nether terrain was updated in the version of MineKhan. New users at this time: RedMonster12, weqwef (aka Guineapigbridge and SpiderScript), Magic4792 New users near this time: Brenda_Puckle In July 2021, RedMonster12 began suggesting blocks to add to MineKhan and uploaded many textures. After sideways torches were added to thingmaker's version of MineKhan, it was updated to 1.0.3. image In August 2021, Luke started to add pistons to his copy of thingmaker's version of MineKhan. New users near this time: multibuilder, novacoding, SpaceSoldier (other usernames: SpaceSoldier248), TayyabR, Aussie Audra September 2021, image: New users: CrackersthehackerYT In October 2021, panoramas were added to the version of MineKhan and it was updated to 1.0.4. image At this time, thingmaker made a post about maps for halloween. The scarecrow was used for the panorama. New users: HB_the_Pencil, SomeBoyPlayz (goldenPvP on yt), Adith image Then, in November 2021, the player model in the version of MineKhan was updated and changing skins was added. New users: Jay2021, Loco, Ironbrodiet

The website for MineKhan

Near December 2021, TNJS (https://replit.com/@tnjs) invited thingmaker to help make a website. The idea was to create a website where people could upload MineKhan maps to share and download them. It was likely that someone told him to ask thingmaker to help create the website for MineKhan maps. So, the website for MineKhan was created by thingmaker and TNJS. The first things added were a way to upload maps. A few weeks later, push notifications were added. Anyone who subscribed would get a notification when a new map was uploaded. Image from this time: New users at this time: goatedkiller, TEXAN7, Candy (possibly wrong time), finn, ThrustBacon, ratrodford1, Therealmushroom New users near this time: Butter (other usernames: Am_Butter, Sharky), Entity_303, Mrbeast In Janurary 2022, an update for redstone had started. After a few days, the server also started to run out of space for users and posts. New users: Trexler180 (other usernames: Trexler), Prytor, Kiro_sama, Zenith So in Feburary 2022, they were moved to a different place. (r____t db to level db) At this time, on Feburary 18, the version of MineKhan was getting very popular and there were many worlds in multiplayer at the same time, so a limit of 5 worlds at the same time was added. New users: repeat In March 2022, there was an idea to create a wiki for MineKhan on the website for MineKhan. So the wiki for MineKhan was created, and it looked really bad at first. On April 1, 2022, the redstone update for the version of MineKhan was released and it was updated to 1.0.5. image On April 14, 2022, thingmaker started to create the first external server for the version of MineKhan, which was called 'Test server'. (https://replit.com/@MineKhan/mkServer) External servers were a way to host a world for the version of MineKhan where the world stayed open almost all the time and the world could be controlled with code. At first, this external server did not have a world, instead, when joining, it asked for a suggestion for the name. Then, on April 18, a world save code was added, and on April 19, portals (teleports to another room) were added. The code for external servers was still bad and it was easy to cheat. (flying, getting any item) New users in April 2022: shubbleYT, eaglermaster (other usernames: Eaglemasterussr), Fire_Fox, sadlyitsnotme, AXL_SENSEI On May 3 2022, whitelist was added to the version of MineKhan and the server. Capes and cloud saves for MineKhan were also added.

(there may have been more that were not logged in) Soon thingmaker's version of MineKhan was discontinued, because the amount of players had decreased a lot.
