Early days of thingmaker's website

The website started as many files and folders on a old computer at somewhere between late 2020 to early 2021. At this time, it was not accessible from the internet as it was local files. The home page and nav bar were copied from w3schools, and the home page showed your programs and webpages. There was a code editor where you could make programs and webpages, which was inspired by KA. This was also when thingmaker started adding things to MineKhan. Some time before December 2020, he downloaded a version of MineKhan (0.7.1) originaly from some time before this. The first thing added was tnt block, which was already in the code. Then portal blocks and obsidian were added, which probably did not do anything. The portal blocks textures copied from another spin-off. Later adding redstone was tried, by adding on and off blocks, then a block called 'buffer' was added which was supposed to behave like a redstone repeater. The redstone blocks didn't work well because it caused errors often, but it stayed. Then textures and new blocks called 'soup' were added to it. At these times, thingmaker often searched for new MineKhan spin-offs to copy things from. Some of the things copied from the spin-offs were fall damage and island world type and biomes (desert and field). Starting near April 2021, thingmaker checked the profile of RedMonster (https://www.khanacademy.org/profile/monsterred/projects) frequently for new things to add to the version of MineKhan. Some of those things were flowers, doors, lanterns, cactus. Some time later, cows and pigs were added which move in random directions and looked bad. On May 8 2021, the version of MineKhan with changes by thingmaker was uploaded to R____t. A few days later, on May 12 2021, the website was also moved onto R____t (https://replit.com/@thingMaker/www) and became accessible from the internet. The website was still very basic. Then, thingmaker wanted to add a login to the website, which was also probably inspired by KA. So it was added and you could change your username and the username showed on the nav bar. On May 24 2021, a server (https://replit.com/@thingMaker/server) was made for the website. A simple login were added to the website and server. The code for the login was copied from somewhere else because thingmaker did not know how to make a login at that time. Websockets were also experimented with because they would allow adding multiplayer to thingmaker's version of MineKhan. In May 2021, multiplayer was added to the version of MineKhan. Posts and comments on posts was added to the server and website. Posts were a part of the website where people could post text. Then, the first post not by thingmaker appeared. (name: Hello, by: Spider, content: Hello)

Some time near this time, thingmaker tried to add breaking blocks in survival mode to the version of MineKhan. The time for breaking grass and dirt was added, which was very fast. Then, adding crack texture was tried. At first, the block textures got messed up. After some time, the crack texture showed up almost correctly because it had z-fighting. It was left like that.

New users near this time: Spider, sup6666, MyHat3, Pillow1007 In June 2021, profile pictures were added to the website and server. Notifications were also added and a way to ban in MineKhan was added to the server. Live comments were also added to posts. A few sounds including grass sound and stone sound were added to the version of MineKhan. On June 10, RedMonster added many more sounds to a fork (https://replit.com/@RedMonster12/minekhan-update) of the version of MineKhan.

The website for MineKhan

Near December 2021, TNJS (https://replit.com/@tnjs) invited thingmaker to help make a website. The idea was to create a website where people could upload MineKhan maps to share and download them. It was likely that someone told him to ask thingmaker to help create the website for MineKhan maps. So, the website for MineKhan was created by thingmaker and TNJS. The first things added were a way to upload maps. A few weeks later, push notifications were added. Anyone who subscribed would get a notification when a new map was uploaded.
